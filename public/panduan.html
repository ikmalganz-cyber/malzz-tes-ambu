Panduan Pembelian

Ikuti langkah berikut untuk melakukan top up game dengan mudah

1

Pilih Produk Game

Buka halaman Produk, temukan game yang ingin kamu top up. Klik pada kartu game untuk melanjutkan ke halaman top up.

2

Masukkan User ID

Masukkan User ID atau ID akun game kamu dengan benar. Cek ID kamu di profil dalam game. Pastikan tidak ada typo karena top up tidak bisa dikembalikan.

3

Pilih Nominal

Pilih paket atau nominal top up yang sesuai. Harga sudah termasuk semua biaya layanan. Nominal yang kamu pilih akan tampil di ringkasan order.

4

Bayar via QRIS

Klik "Lanjut Bayar" dan scan QR code menggunakan e-wallet kamu: GoPay, OVO, Dana, ShopeePay, atau mobile banking apapun yang support QRIS. Bayar sesuai nominal yang tertera.

5

Selesai!

Setelah bayar dikonfirmasi, top up masuk ke akunmu otomatis. Status real-time tersedia di halaman pembayaran. Cek juga di halaman status menggunakan ID Order.

Pertanyaan Umum

Berapa lama proses top up?

Top up biasanya selesai dalam 1-5 menit setelah pembayaran dikonfirmasi. Kondisi server game yang sibuk bisa mempengaruhi waktu proses, maksimal 15 menit.

Metode pembayaran apa yang tersedia?

Kami menerima QRIS universal yang bisa discan dari semua aplikasi e-wallet dan mobile banking: GoPay, OVO, Dana, ShopeePay, LinkAja, BRI, BCA, BNI, Mandiri, dan semua bank yang support QRIS.

Bagaimana jika top up tidak masuk?

Cek dulu status order di halaman Cek Status. Jika statusnya pending lebih dari 15 menit setelah pembayaran dikonfirmasi, segera hubungi kami dengan ID order kamu.

Apakah ID game saya aman?

Kami hanya menggunakan ID game untuk keperluan pengiriman top up. Data kamu tidak pernah disimpan untuk keperluan lain atau disebarkan ke pihak manapun.

Bisakah order dibatalkan atau di-refund?

Sayangnya tidak. Setelah pembayaran dikonfirmasi dan proses berjalan, order tidak dapat dibatalkan. Pastikan ID dan nominal sudah benar sebelum bayar.

Berapa batas waktu pembayaran?

Kamu punya waktu 15 menit untuk menyelesaikan pembayaran setelah QR code muncul. Jika lewat dari batas waktu, kamu perlu membuat transaksi baru.
